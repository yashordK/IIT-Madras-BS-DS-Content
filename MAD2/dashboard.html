💡 Study Tips
🎯

Focus on Quirks

Graded assignments love JS quirks — typeof null === "object", default .sort() on numbers, var leaking out of blocks.

💻

Code in Browser Console

Open DevTools → Console and test every code snippet you read. Typing it yourself beats reading it 3x over.

🧠

Take Each Quiz

Every notes file has a 10-question quiz at the end. Answer without looking — checking recall is better than re-reading.

🔗

Connect the Concepts

Async → Event Loop → Promises → async/await is one logical chain. Don't memorize in isolation — understand the "why".

📊

Master map/filter/reduce

These three cover most array manipulation problems in assignments. Know what each returns and how to chain them.

⚡

Prof's Rules to Remember

Always use semicolons. Avoid var. Prefer const > let. Stick to English identifiers. Arrow functions for short callbacks.
