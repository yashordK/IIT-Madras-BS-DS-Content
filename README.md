## Coverage

### ✅ Week 1 — JavaScript Fundamentals
- L1: MAD1 review + what MAD2 covers (MVC, Flask, REST recap)
- L2: JS history — Brendan Eich, ECMAScript, ES6, polyfills, Babel, ASI
- L3: Data types, `typeof` quirks, `null` vs `undefined`
- L4: `var` vs `let` vs `const` — scoping, const in loops
- L5: Control flow (`for`, `for...in`, `for...of`), 3 function styles (declaration, expression, arrow)
- L6: DOM API — `getElementById`, `innerHTML`, `addEventListener`, `setTimeout`

### ✅ Week 2 — Advanced JavaScript
- L1: Arrays, iterables, `map` / `filter` / `find` / `reduce`, generators
- L2: Iteration patterns, spread operator, array sort with comparator, destructuring
- L3: ES6 modules (`import`/`export`), npm, object literals, `this` keyword
- L4: Getters/setters, `call`/`apply`/`bind`, prototypes, ES6 classes & inheritance
- L5: Call stack, event loop, callbacks, Promises, `async`/`await`
- L6: JSON — `stringify`, `parse`, REST API data transfer

## Why This Exists

The IITM BS lectures are good — but they're long, and re-watching them before a graded assignment isn't efficient. These notes give you:

- **The exact concepts** covered in each lecture (from actual transcripts, not guesswork)
- **Code examples** you can copy and run
- **Prof's own rules** highlighted — what to use, what to avoid
- **10-question quizzes** per week to test recall, not just recognition
- **Tables for comparison** — `var` vs `let` vs `const`, callback vs Promise vs async/await, etc.

Built to help you go from "I watched the lecture" to "I can answer the graded assignment" as fast as possible.
